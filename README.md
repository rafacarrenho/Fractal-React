Clone the project and access the folder

```bash
$ git clone https://github.com/rafacarrenho/Fractal-React && cd Fractal-React
```

Follow the steps below

```bash
# Install the dependencies
$ npm install

# Start the project
$ npm start
```

```bash

$ yarn install

# Start the project
$ yarn start
```
